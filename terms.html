2 Description of Service

LAM is a Progressive Web Application (PWA) that provides logistics and business management tools including fleet management, trip tracking, invoice generation, GST compliance tools, driver management, and financial reporting. The core application operates entirely on the user's device (offline-first) using browser IndexedDB storage.

Optional cloud sync features are available and require the user to configure their own Supabase project. Optional payment features require a Razorpay account.
